Transformador de texto a texto encriptado

Este método de encriptado funciona con el método de Fernet,
se usará el archivo contraseña.txt para conseguir la información a desencriptar,
se creará un archivo desencriptado.txt en la carpeta seleccionada del dispositivo
en la cual estará el texto descriptado y su clave.
Si va a usar esta información para algo importante ¡no la comparta con nadie!
